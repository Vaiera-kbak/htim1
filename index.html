Введение в HTML

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц.

HTML описывает структуру веб-страницы с помощью элементов, которые браузеры отображают как заголовки, абзацы, ссылки и т.д.

Пример простой HTML-страницы:

<!DOCTYPE html>
<html>
<head>
    <title>Моя первая страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это моя первая HTML-страница.</p>
</body>
</html>

Основы HTML

Каждый HTML-документ состоит из элементов, которые обозначаются тегами.

Структура HTML-документа:

  • <!DOCTYPE html> - объявление типа документа
  • <html> - корневой элемент страницы
  • <head> - содержит метаинформацию о странице
  • <body> - содержит видимое содержимое страницы

Основные HTML-элементы

Заголовки:

<h1>Заголовок 1 уровня</h1>
<h2>Заголовок 2 уровня</h2>
<h3>Заголовок 3 уровня</h3>
<h4>Заголовок 4 уровня</h4>
<h5>Заголовок 5 уровня</h5>
<h6>Заголовок 6 уровня</h6>

Абзацы и ссылки:

<p>Это абзац текста.</p>
<a href="https://example.com">Это ссылка</a>

Изображения:

<img src="image.jpg" alt="Описание изображения">

HTML-формы

Формы используются для сбора пользовательского ввода.

<form action="/submit" method="post">
    <label for="name">Имя:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <label for="message">Сообщение:</label>
    <textarea id="message" name="message"></textarea>
    
    <button type="submit">Отправить</button>
</form>
